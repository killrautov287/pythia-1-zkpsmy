## Installation

- Install [Circom2](https://docs.circom.io/getting-started/installation/) (rust version)
- Build

```sh
$ yarn build
```

## Test

```sh
$ yarn test
$ test:circuits
$ test:verifier-js
$ test:verifier-contract
$ test:prover-js
```
